Lua基本语法

变量
LUA_INIT=”filename” 运行lua脚本时首先执行
arg[1]=”a” arg是存储命令行参数的table，参数从1开始，0,-1…为命令行其他内容
local a 声明局部变量
local a=a 用全局变量创建同名局部变量，可以加速块内访问速度

类型
nil 空
boolean 布尔,只有false和nil为假，true及其他都算真
number 数字，默认为32位双精度浮点数
string 字符串，不可变，支持任意二进制数据和任意大小
userdata 自定义类型
function 函数
thread 线程
table 表
4e+4,5e-1 支持科学计数法
\n 转义字符，换行
\r 转义字符，回车
\t 转义字符，tab
\ 转义字符，\
\’ 转义字符,’
\” 转义字符,”
\ 三个十进制数字，引用ASCLL
[[]],[===[ ]===] 0或任意数量等号界定的格式字符串，内部字符不会被解析

Table数据结构

table的声明

不能显式声明table，可以通过构造表达式声明

table是永远匿名的删除其引用（指针）不会删除table，当没有引用时，对table进行垃圾回收。

table的引用

a.k == a[k]

#a 表示a的大小或最后一个元素的索引

table默认从1开始计数

函数

函数语法

  1. 只有一个参数且是字符串或table可以不加括号
  2. 冒号操作符 o.f(o,b) == o:f(b)
  3. 支持多重返回值 return a,b
  4. 强制全部返回 return f()
  5. 强制只返回第一个 (f())
  6. unpack() 接受数组参数，并从1开始返回数组的元素
  7. 函数定义，接收变长参数 function f(…) end
  8. 所有参数在数组‘…’里
  9. 不支持具名实参传入(a=’a’,b=’b’)，但是可以传入一个table,{a=’a’,b=’b’}
  10. 所有函数都是匿名的，函数名只是函数指针
  11. 匿名函数 function (x) return x end
  12. 支持高阶函数
  13. closure闭合函数，函数加上非局部变量
  14. 尾调用作用消除，return g() 相当于 goto g
函数或操作符名 解释
type() 返回目标类型，字符串
+，-，×，/，%，^ 加减乘除取模指数，对字符串会进行强转成数字
<,>,<=,>=,==,~= 关系运算符，返回true或false，nil只与自身相等
and,or,not 逻辑运算，有短路特性
A and B or C 类似A?B:C,但B不能为假(false或nil)
x,y=y,x 多重赋值可以交换变量
.. 字符串连接操作
tonumber() 强转为数字
tostring() 强转为字符串
#string 获得字符串长度
table.maxn() 返回最大正索引数

杂项

  1. 代码不需要分隔符，换行和分号都没有作用
  2. 标识符以字母或下划线开头，不能以数字开头
  3. 单行注释： –something
  4. 多行注释： –[[A,B,C//\n –]]
  5. 代码测试技巧：在多行注释前一个加上一个-取消注释，即 –[[ -> —[[
  6. 脚本头格式 #!/usr/local/bin/lua

解释，编译
lua -i a.lua 在执行完脚本之后将进入交互模式
lua -e “print()” 单行执行命令
lua -l a 加载库文件
dofile(‘filename’) 交互式中执行脚本
loadfile(‘filename’) 加载脚本，编译成函数后返回这个函数
loadstring(string) 将字符串作为lua代码执行，从内存中获得代码
loadlib() 加载动态链接库

assert(f(),”string”)

若f()返回nil或false则返回string，若f()返回true则返回f()的值

协同程序 coroutine

四种状态

  1. suspended 挂起
  2. running 执行
  3. dead 死亡
  4. normal 正常

函数
coroutine.create(匿名函数) 创建一个协同函数
coroutine.status(函数名) 返回函数状态
coroutine.resume(实参…) 使函数执行
coroutine.yield(函数名，返回值…) 挂起函数

设计模式

  1. 消费者驱动，处理数据的函数唤醒生产者，生产者产生数据后挂起并把数据传递给消费者函数
  2. 非抢先多线程，为每个任务创建一个线程，当一个线程没有数据可处理时把控制权交给调度函数，由调度函数唤起另一个线程
