默认按键

快捷键 作用
Window manager control Awesome控制
MOD4 + CTRL + R 重启 Awesome
MOD4 + Shift + Q 退出 Awesome
MOD4 + Enter 终端模拟器
MOD4 + W 主菜单
MOD4 + R 运行程序
MOD4 + X 执行Lua代码
Clients 窗口控制
MOD4 + M 最大化
MOD4 + N 最小化
MOD4 + Ctrl + N 恢复最小化
MOD4 + F 全屏
MOD4 + Shift + C 退出程序
MOD4 + t 放在最前
