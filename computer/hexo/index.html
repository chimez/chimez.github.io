Hexo的安装

  1. 安装npm： pacman -S npm
  2. 配置npm： npm config
  3. 安装hexo： npm install hexo
  4. 初始化hexo： hexo init
  5. 切换到目录： cd
  6. 安装支持： npm install
  7. 配置hexo： vim _config.yml

Hexo 命令

命令 作用
source : public/_posts/title.md 文章的目录
hexo init (folder) 初始化
hexo new [layout] (title) 创建新文章、页面
hexo generate 编译生成网页
hexo g 同上
hexo server 开启本地服务器预览
hexo s 同上
hexo publish [layout] (filename) 将草稿发布
hexo deploy 上传到服务器
hexo d 同上
