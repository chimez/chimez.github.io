tikz 画图

基本使用方式

  1. 去这里抄代码： http://www.texample.net/tikz/examples/
  2. \usepackage{tikz}
  3. 默认长度单位cm
  4. 默认聚焦点为左下角
  5. 命令式：\tikz \draw
  6. 每句后有分号

路径

缩写格式：

  • \draw=\path[draw] 画线
  • \fill=\path[fill] 填充
  • \clip=\path[clip]
  • \filldraw=\path[fill,draw] 画线并填充
  • \shade=\path[shade] 阴影
  • \shadedraw 阴影

简单做图

图形 语句
直线,坐标–坐标 \draw (0,0)–(2,0)–(3,1);
闭合直线，坐标–cycle \draw (0,0)–(2,0)–(3,1)–cycle;
圆，圆心，半径 \draw (0,0) circle (10pt);
椭圆，圆心，长半轴and短半轴 \draw (0,0) ellipse (20pt and 10pt);
矩形，对焦点坐标 \draw (0,0) rectangle (2,1);
网格，小格步长，左下坐标，x长度，y高度 \draw[step=5pt] (0,0) grid (30pt,20pt);
网格步长选项 step,xstep,ystep;
圆弧,起始点，角度范围:半径 \draw (0,0) arc (0:135:1);
椭圆弧，起始点，角度范围：长半轴and短半轴 \draw (0,0) arc (0:270:1 and 0.6);
抛物线，起点，顶点 \draw (0,0) parabola (-1,2);
增强型抛物线,起点，顶点，终点 \draw (0,0) parabola bend (1,1) (2,0);
两个控制点的贝塞尔曲线，起点..控制点1and控制点2,若只写一个控制点，第二个与第一个相同 \draw (0,0) .. controls (1,1) and (2,1) .. (2,0);
sin，cos线连接点，起点，终点 \draw (0,0) sin (1,1);

语句选项

语句选项 解释
draw 画线
line width=1pt 线宽
thin, very thin, ultra thin, thick, very thick, … 预定义线宽
dash pattern=on 2pt off 3pt on 4pt 手动设置线条形状，on、off表示画线不画线的长度，重复出现
dash phase=1pt 设置第一个虚线缩短的长度
solid,dashed,densely dashed, loosely, loosely dashed, dooted， debsely dooted, loosely dotted 预定义线型
color=颜色 设置线色
draw=颜色 设置线色
fill=green!20 填充颜色
opacity=0.5 设置透明度
rounded corners=10pt 设置拐点圆角半径，默认为0
scale=0.3 放大缩小倍数
double 画双线
double distance=1pt 双线间距离
left, right, top, bottom, inner, outer, ball 阴影的渐变方向

backgrounds 显示绘图边框

坐标格式 解释
(x,y) 直角坐标

函数绘图

函数做图的四种方式：

  • plot [选项] coordinate {(x,y),(x,y),…}
  • plot [选项] file {文件名}
  • plot [选项] 坐标表达式
  • plot [选项] function {函数 gnuplot}

选项参数

选项参数 解释
smooth 光滑曲线
smooth cycle 光滑封闭曲线
tension=0.55 曲线松紧度
variable=\t 指定变量，默认x
samples=25 制定样本点个数
domain=-5:5 绘图区间
samples at={x1,x2,x3,…} 制定样本点值

文件格式：

  1. 每行两个数字，用空格隔开
  2. #%为注释

支持的数学运算：

  • +, -, *, /, ^
  • abs{}, exp{}, ln, pow, sqrt, sin, cos, tan, sec, pi, …
  • x r -> 弧度制到角度制
