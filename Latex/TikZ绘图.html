TikZ绘图

基本使用方式

  • 去这里抄代码
  • \usepackage{tikz}引入宏包
  • 默认长度单位cm
  • 默认聚焦点为左下角
  • \tikz \draw 命令式
  • 每句后有分号

绘图环境

\begin{tikzpicture}\end{tikzpicture} 主环境
\begin{scope}[]\end{scope} 局部参数环境,嵌套在tikzpicture环境里
\usetikzlibrary{scopes}{[]} 扩展：用大括号代替环境

基本元素

\draw=\path[draw] 画线
\fill=\path[fill] 填充
\clip=\path[clip]  
filldraw=\path[fill,draw] 画线并填充
\shade=\path[shade] 阴影
\shadedraw 阴影
\tikz\path[draw,thick] (1,1)--(2,2)--(3,2)--cycle  

简单做图

\draw (0,0)--(2,0)--(3,1); 直线,坐标--坐标
\draw (0,0)--(2,0)--(3,1)--cycle; 闭合直线，坐标--cycle
\draw (0,0) circle (10pt); 圆，圆心，半径
\draw (0,0) ellipse (20pt and 10pt); 椭圆，圆心，长半轴and短半轴
\draw (0,0) rectangle (2,1); 矩形，对焦点坐标
\draw[step=5pt] (0,0) grid (30pt,20pt); 网格，小格步长，左下坐标，x长度，y高度
step,xstep,ystep; 网格步长选项
\draw (0,0) arc (0:135:1); 圆弧,起始点，角度范围:半径
\draw (0,0) arc (0:270:1 and 0.6); 椭圆弧，起始点，角度范围：长半轴and短半轴
\draw (0,0) parabola (-1,2); 抛物线，起点，顶点
\draw (0,0) parabola bend (1,1) (2,0); 增强型抛物线,起点，顶点，终点
\draw (0,0) .. controls (1,1) and (2,1) .. (2,0); 两个控制点的贝塞尔曲线，起点..控制点1and控制点2,若只写一个控制点，第二个与第一个相同
\draw (0,0) sin (1,1); sin，cos线连接点，起点，终点

语句选项

draw 画线
line width=1pt 线宽
thin, very thin, ultra thin, thick, very thick, ... 预定义线宽
dash pattern=on 2pt off 3pt on 4pt 手动设置线条形状，on、off表示画线不画线的长度，重复出现
dash phase=1pt 设置第一个虚线缩短的长度
color=颜色 设置线色
draw=颜色 设置线色
fill=green!20 填充颜色
opacity=0.5 设置透明度
rounded corners=10pt 设置拐点圆角半径，默认为0
scale=0.3 放大缩小倍数
double 画双线
double distance=1pt 双线间距离
left, right, top, bottom, inner, outer, ball 阴影的渐变方向

backgrounds 显示绘图边框

坐标的表示

(x,y) 直角坐标
(\(\theta\):r) 极坐标
+(,) 由上个当前点进行偏移
++(,) 由上个当前点进行偏移，并设为新的当前点
(intersection of 0,0--1,1 and 0,1--1,0) 两直线交点

坐标变换

xshift=2pt x方向平移
yshift-2pt y方向平移
shift={(5pt,5pt)} 向量平移
rotate=30 旋转30度
scale=1.1 整体缩放
xscale=2 x方向缩放
yscale=2 y方向缩放
xslant=1 x方向倾斜
yslant=2 y方向倾斜

重复

\foreach 变量 in {值表} {命令} \foreach \x in {1,2,3} { \draw (\x,0) circle (3pt)}

  • 变量名前加反斜线 \x
  • 可使用多个变量

文字标注

coordinate

\coordinate [label=角度：\(标注\)] (标记) at (x,y) \coordinate [label=45:\(adss\)] (a) at (1,1) \coordinate [label=-45:\(vvdewf\)] (b) at (3,1) \draw (a)--(b)

node

node [anchor=north] {$$} anchor:指定文本排放相对位置 north, south, west, east, below, above, left, right \node [选项 ] (名称) at (x,y) [选项] {text}

函数绘图

plot [选项] coordinate {(x,y),(x,y),...} 点绘图
plot [选项] file {文件名} 文件绘图
plot [选项] 坐标表达式 坐标绘图
plot [选项] function {函数 gnuplot} 函数绘图

选项参数

smooth 光滑曲线
smooth cycle 光滑封闭曲线
tension=0.55 曲线松紧度
variable=\t 指定变量，默认x
samples=25 制定样本点个数
domain=-5:5 绘图区间
samples at={x1,x2,x3,...} 制定样本点值
  • 文件格式:每行两个数字，用空格隔开
  • #%为注释
  • 支持的数学运算

+, -, *, /, ^ abs{}, exp{}, ln, pow, sqrt, sin, cos, tan, sec, pi, ... x r -> 弧度制到角度制

标记选项

\usetikzlibrary{plotmarks} 引入宏包
mark=* 标记形状
mark size=1ex 标记大小
mark options={} 标记属性
color=red 标记属性->颜色
rotate=20 标记属性->方向旋转
ycomb y轴垂线
xcomb x轴垂线
polar comb
only marks 不连线只有点
