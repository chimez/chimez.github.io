1 基本语法

1.1 基本程序示例

在文件 hello.pl 中输入下面的内容， 并运行 perl hello.pl 

#!/usr/bin/perl
use strict;
use warning;

print "Hello, world";

1.2 变量

一共有三种变量类型： $scalar, @array, %hash

1.2.1 scalar

  • undel : 类似 python 的 None
  • 数字，不区分整数和浮点
  • 字符串
  • 引用

  • bool: 只有 undel, 0, "", "0" 才是 false, 其余的都认为是 true 

    my $animal = "camel";
my $num = 42;

print "$animal are", $num * 2, "s" . "\n";

1.2.2 array

  • 使用圆括号，元素最后可以有逗号
  • 引用元素用美元号和方括号
  • $#arr 是最后一个元素的序号
  • @arr 直接被解释为元素的个数
  • 使用 [1..2] 做引用
  • 参数列表在 @ARGV 里
my @arr = ("str", 12, 1.2,);

print $arr[0], $arr[1]; # -> str12
print @arr;             # -> str121.2
print $arr[$#arr];      # -> 1.2
print @arr[0..1];       # -> str12

1.2.3 hash

  • 系统环境变量在 %ENV 里
my %fruit_color_1 = ("apple", "red", "banana", "yellow");

# or

my %fruit_color = (
    apple  => "red",
    banana => "yellow",
    );

print $fruit_color{"apple"};           # gives "red"

# keys and values
my @fruits = keys %fruit_color;
my @colors = values %fruit_color;

1.2.4 引用

  • \@arr : 通过转义获得引用
  • @{\@arr} : 通过大括号解引用
  • (\@arr) -> [1] : 通过箭头在解引用后获得元素
  • 方括号创建的是针对 array 的一个引用，这一点会用于创建嵌套数据
my @arr = (1, 2, 3);
my @arr2 = (\@arr, 2);
my $arr3 = [1, 2, 3];

print ${ $arr2[0] }[0];
print $arr2[0] -> [0];
print @{ $arr3 };

my %h = (
	k1 => 1,
	k2 => [2,3],
	k3 => [
		kk1 => 4,
		kk2 => 5,
	]
);

print @{ $h{"k2"}};

    ...

    ...

1.3.3 foreach

foreach (@array) {
    print "This element is $_\n";
}

print $list[$_] foreach 0 .. $max;

# you don't have to use the default $_ either...
foreach my $key (keys %hash) {
    print "The value of $key is $hash{$key}\n";
}

foreach my $i ( 0 .. 10) {
    print $i;
}

类似 continue, break , 可以标号跳转

    }

}

1.5 subroutines

  • 相当于函数
  • 传参数是引用而不是值，所以如果在函数里修改输入参数，会真的改变它
  • 参数列表是 @_
  • shift 默认就是 shift @_
  • 推荐做法是用 shift 一个一个的读参数
  • 或者 my ($a, $b) = @_ 一次性构造参数
  • 不要 my $a = @_[0] 这样用
  • 使用 return (...) if wantarray; 可以返回列表
sub square {
     my $num = shift;
     my $result = $num * $num;
     return $result;
 }

$sq = square(8);

2.4 基本文件操作

2.4.1 读写文件

  • open 会开启一个文件句柄
  • close 会关闭它
  • 文件在超出作用域后才会自动关闭，所以全局变量必须手动关闭
  • <fi> 尖括号包围可以读取文件
  • readline $fh 可以读取一行
  • 三个默认文件 STDIN, STDOUT, STDERR
open(my $in,  "<",  "input.txt")  or die "Can't open input.txt: $!";
open(my $out, ">",  "output.txt") or die "Can't open output.txt: $!";
open(my $log, ">>", "my.log")     or die "Can't open my.log: $!";

my $line  = <$in>;
my @lines = <$in>;

while (<$in>) {     # assigns each line in turn to $_
    print "Just read in this line: $_";
}

close $in or die "$in: $!";

2.5 系统调用

  • 使用反引号的命令，标准输出保存在指定的变量和 $ ? 中
  • 使用 system 的调用，标准输出直接打印，退出码存在指定变量和 $? 中
  • 返回码保存在高8位，即 $? >>=8
  • perl 程序的返回码用 exit 123 指定
  • chdir "path" 可以改路径
my $p1 = `ls`;
print $? >> 8, "\t", $p1, "\n";
my $p2 = system "ls";
print $? >> 8, "\t", $p2 >> 8, "\n";

exit 3;

3 问题解决

3.1 手写一个进度条

参考： progress bar in command line perl script

  1. local $| = 1; 开启 autoflush
  2. "\b" 移动光标到前一个字符处
use strict;
use warnings;

use Time::HiRes qw(usleep);
local $| = 1;

my @nums = 1 .. 20;

foreach my $c (@nums) {
  print "$c";
  usleep(100000);
  print ("\b" x length($c));
}
print "\n";

3.2 单行脚本：管道输入

参数使用 -n 表示循环读入每一行, 运行结果用 print 

result=`ps u | perl -ne '($user,$pid)=/([a-z]+) +(\d+) /; print "${user}:${pid},"'`
echo $result
