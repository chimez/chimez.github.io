1.1.2 自动求导

自动求导是神经网络的中心. pytorch 的自动求导是在运行中定义的, 即它会记录所进行的求值操作, 并在完成所有计算后调用求导函数, 自动沿着记录的这些函数栈进行求导. 在进行求值时, 可以指定不记录某些求值过程, 也可以停止进行记录.

  1. 通过设置张量的属性 .requires_grad 为 True, 即可开始进行记录. 使用 .requires_grad_(True) 方法,可以随时改变是否进行记录.
  2. 记录结束后使用 .backward() 方法, 就可以计算自动求导, 并将结果储存在原始张量(自变量)的 .grad 属性中.
  3. 如果张量是标量, 调用 .backward() 不用指定任何参数, 否则要指定张量的形状.
  4. 调用 .detach() 方法, 会停止记录接下来的所有操作.
  5. 只是不想记录某些操作的话, 使用 with torch.no_grad(): 函数.
  6. 自动求导的实现使用了 torch.Function 类, 每个由函数自动创建的张量都有一个 .grad_fn 属性来指向相关的这个 Function, 而那些由用户自己创建的张量, 这个属性值为 None.
  1. 例子:一个函数的自动求导,标量情况

    下面的例子计算了:\[\frac{d(out)}{dx}= \frac{d3(x+2)^{2}}{dx}= \left(\begin{matrix} 4.5& 4.5\\ 4.5& 4.5 \end{matrix}\right) \], \(x=I\) 

    import torch
x = torch.ones(2, 2, requires_grad=True)
print("一个开始记录的张量,可以看到它的属性值为 True:\nx\t=\t{}".format(x))
y = x + 2
print("由函数产生的张量,它有一个指向这个函数的属性:\ny\t=\tx + 2\t=\t{}".format(y))
z = y * y * 3
out = z.mean()
print("再加入一些操作:\nout\t=\t=\t<3y^2>\t=\t{}".format(out))
print("进行自动求导")
out.backward()
print("得到:\n $\\frac{{d(out)}}{{dx}}$ = {}".format(x.grad))
  2. 例子:自动求导, 指定自变量
    import torch
x = torch.randn(3, requires_grad=True)
print("初始值:\n{}".format(x))
y = x * 2
while y.data.norm() <= 1000:
    y = y * 2
    print("乘上n次:\n{}".format(y))
    gradients = torch.tensor([0.1, 1.0, 0.0001], dtype=torch.float)
    print("需要指定自变量:\n{}".format(gradients))
    y.backward(gradients)
    print("自动求导:\n{}".format(x.grad))

1.1.3 神经网络

神经网络使用 torch.nn 模块, 这个模块建立在自动求导机制之上. 神经网络的典型训练过程为:

  1. 定义一个有一些可学习的参数(或者叫权重)的神经网络.
  2. 在输入的数据集上迭代
  3. 通过网络处理输入
  4. 计算损失(即输出到目标差多远)
  5. 向神经网络的参数传播回去自动微分值
  6. 更新神经网络的参数,典型使用的公式为: 权重=权重-学习率*微分值
  1. 定义神经网络

    只需要继承 nn.Module 类,并在 __init__ 里定义所有的层, 以及在 forward 里定义向前传播的计算规则即可, 向后传播由自动微分自动完成. 

    import torch
import torch.nn as nn
import torch.nn.functional as F


class Net(nn.Module):

    def __init__(self):
	super(Net, self).__init__()
	# 1 input image channel, 6 output channels, 5x5 square convolution
	# kernel
	self.conv1 = nn.Conv2d(1, 6, 5)
	self.conv2 = nn.Conv2d(6, 16, 5)
	# an affine operation: y = Wx + b
	self.fc1 = nn.Linear(16 * 5 * 5, 120)
	self.fc2 = nn.Linear(120, 84)
	self.fc3 = nn.Linear(84, 10)

    def forward(self, x):
	# Max pooling over a (2, 2) window
	x = F.max_pool2d(F.relu(self.conv1(x)), (2, 2))
	# If the size is a square you can only specify a single number
	x = F.max_pool2d(F.relu(self.conv2(x)), 2)
	x = x.view(-1, self.num_flat_features(x))
	x = F.relu(self.fc1(x))
	x = F.relu(self.fc2(x))
	x = self.fc3(x)
	return x

    def num_flat_features(self, x):
	size = x.size()[1:]  # all dimensions except the batch dimension
	num_features = 1
	for s in size:
	    num_features *= s
	return num_features


net = Net()
print(net)

    神经网络的参数可以通过 net.parameters() 获得. 

    params = list(net.parameters())
print(len(params))
print(params[0].size())  # conv1's .weight

    现在给一个 32x32的输入,可以看到它通过这个神经网络后的输出值大小: 

    input = torch.randn(1, 1, 32, 32)
out = net(input)
print(out)

    计算自动微分的方法很简单,只需要先通过 net.zero_grad() 方法将微分初值设为随机值, 再用 out.backward() 方法计算自动微分即可. 

    net.zero_grad()
out.backward(torch.randn(1, 10))

    注意: torch.nn 只支持小批量的样本, 例如 nn.Conv2d 的参数是 nSamples x nChannels x Height x Width 的四维张量. 如果只有一个样本, 可以用 input.unsqueeze(0) 创建一个假的批量维度.

  2. 损失函数

    损失函数接受输出结果和目标这两个参数, 计算结果到目标的距离. 在 torch.nn 包里有各自各样的损失函数, 其中一个是 nn.MSELoss 用来计算均方根误差. 

    output = net(input)
target = torch.randn(10)  # a dummy target, for example
target = target.view(1, -1)  # make it the same shape as output
criterion = nn.MSELoss()

loss = criterion(output, target)
print(loss)

    如果使用 loss.backward() 整个图都会关于损失函数进行微分 ,因为现在被微分的栈是:

    input -> conv2d -> relu -> maxpool2d -> conv2d -> relu -> maxpool2d -> view -> linear -> relu -> linear -> relu -> linear -> MSELoss -> loss

  3. 反向传播

    现在所需要做的事情只是先清零 net.zero_grad(), 再调用 loss.backward(), 就得到了自动微分值. 如果不进行清零的话, 自动微分值会累加. 

    net.zero_grad()     # zeroes the gradient buffers of all parameters
print('conv1.bias.grad before backward')
print(net.conv1.bias.grad)

loss.backward()
print('conv1.bias.grad after backward')
print(net.conv1.bias.grad)
  4. 更新权重

    最简单的更新规则是随机梯度下降(SGD), 新权重=旧权重-学习率*自动微分值, 实现为: 

    learning_rate = 0.01
for f in net.parameters():
    f.data.sub_(f.grad.data * learning_rate)

    常用的一些更新规则实现在了 torch.optim 包里, 使用步骤为 创建优化器->清零->计算损失函数->反向传播->优化器.下一步(). 下面是一个使用的例子: 

    import torch.optim as optim

# create your optimizer
optimizer = optim.SGD(net.parameters(), lr=0.01)

# in your training loop:
optimizer.zero_grad()   # zero the gradient buffers
output = net(input)
loss = criterion(output, target)
loss.backward()
optimizer.step()    # Does the update

  2. 定义神经网络
    import torch.nn as nn
import torch.nn.functional as F

class Net(nn.Module):
    def __init__(self):
	super(Net, self).__init__()
	self.conv1 = nn.Conv2d(3, 6, 5)
	self.pool = nn.MaxPool2d(2, 2)
	self.conv2 = nn.Conv2d(6, 16, 5)
	self.fc1 = nn.Linear(16 * 5 * 5, 120)
	self.fc2 = nn.Linear(120, 84)
	self.fc3 = nn.Linear(84, 10)

    def forward(self, x):
	x = self.pool(F.relu(self.conv1(x)))
	x = self.pool(F.relu(self.conv2(x)))
	x = x.view(-1, 16 * 5 * 5)
	x = F.relu(self.fc1(x))
	x = F.relu(self.fc2(x))
	x = self.fc3(x)
	return x

net = Net()
  3. 定义损失函数和优化器
    import torch.optim as optim

criterion = nn.CrossEntropyLoss()
optimizer = optim.SGD(net.parameters(), lr=0.001, momentum=0.9)
  4. 训练神经网络
    for epoch in range(2):  # loop over the dataset multiple times

    running_loss = 0.0
    for i, data in enumerate(trainloader, 0):
	# get the inputs
	inputs, labels = data

	# zero the parameter gradients
	optimizer.zero_grad()

	# forward + backward + optimize
	outputs = net(inputs)
	loss = criterion(outputs, labels)
	loss.backward()
	optimizer.step()

	# print statistics
	running_loss += loss.item()
	if i % 2000 == 1999:    # print every 2000 mini-batches
	    print('[%d, %5d] loss: %.3f' %
		  (epoch + 1, i + 1, running_loss / 2000))
	    running_loss = 0.0

print('Finished Training')

2 TensorFlow

2.1 基本工作流程

2.1.1 原理描述

  1. 总体上的过程分两步:先定义session的内容(图),再运行session
  2. 定义会话:会话控制图(graph)的计算过程,将图发送给CPU设备进行计算,再收回计算结果
  3. 图是指一系列顺序的算符(operation)运算,算符作用在一个张量(tensor,多维数组)上得到一个新的张量,一系列算符作用运算完之后返回结果给会话
  4. 每个图都从源算符(常数算符)开始,在会话里执行的是最后一个算符,同时指定了返回值
  5. 变量(variable)用来在运行过程中保存状态和临时变量

2.1.2 执行顺序

input1 = tf.constant(3.0)
input2 = tf.constant(2.0)
input3 = tf.constant(5.0)
intermed = tf.add(input2, input3)
mul = tf.mul(input1, intermed)

with tf.Session():
  result = sess.run([mul, intermed])
  print result
  sess.run([output], feed_dict={input1:[7.], input2:[2.]})
