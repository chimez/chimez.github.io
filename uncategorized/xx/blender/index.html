由标准几何体生成

  1. 创建一个标准几何体
  2. 综合利用挤出(E),环切(Ctrl+R),切割(K),创建面(F)

仿照三视图创建

  1. 添加底图
  2. 创建一个平面,删除三个边得到一条线
  3. 用挤出(E)由线生成点,进而画出半个轮廓
  4. 使用修改器-镜像获得另一半
  5. 用挤出(E)由正视图形成立体,结合环切(Ctrl+R)进行大致轮廓调整 ## 操作快捷键

FAQ

如何设置底图

右侧栏 -> 背景图(最好是正方形) -> 未设置 -> 打开 -> 轴向

如何添加修改器

最右侧属性 -> 修改器(扳手) -> 添加修改器

  1. 修改器 -> 表面细分 与 T -> 平滑顶点
  2. 镜射(镜像)

如何设置中文

  1. 打开上方 file -> User Preferences
  2. 勾选 System 下 International Fonts 条目
