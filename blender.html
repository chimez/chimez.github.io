1 Python API

1.1 由文本读取点数据创建曲线

#Blender Python

import bpy
#polyline.points.add(len(coords))

def file_to_pointlist(filename):
    '''读取文本文档，返回列表'''
    with open(filename, 'r') as fi:
	coords=fi.readlines()

    def clinetomath (text):
	'''解析一行文本返回第一个数字和剩下的文本'''
	n=text.find('e')
	if n==-1:
	    return -1,-1
	a=float(text[:n])
	b=float(text[n+1:n+4])
	return a*10**b,text[n+5:]

    #将全部数字储存为一维列表
    all_number=[]
    for coord in coords:
	num,text=clinetomath(coord)
	while text!=-1:
	    all_number.append(num)
	    num,text=clinetomath(text)

    #将一维列表变成二维
    list=[]
    for i in range(len(all_number)):
	if i%3==0:
	    list.append([all_number[i],all_number[i+1],all_number[i+2]])

    return list

def create_curve(name,list):
    '''创建混合曲线'''
    curveData = bpy.data.curves.new(name, type='CURVE') #创建曲线数据
    curveData.dimensions = '3D' #曲线模式
    curveData.resolution_u = 2  #U向表面精度
    curveData.fill_mode = 'FULL'    #填充模式
    curveData.bevel_depth = 0.001   #倒角深度

    polyline = curveData.splines.new('NURBS')   #创建曲线
    for i in range(len(list)):  #导入点坐标
	x,y,z=list[i]
	polyline.points.add(1)
	polyline.points[i].co = (x,y,z, 1)

    curveOB = bpy.data.objects.new(name, curveData) #创建物体
    curveOB.select = True   #选择物体

    scn = bpy.context.scene #设置场景信息
    scn.objects.link(curveOB)   #在场景中添加物体
    scn.objects.active = curveOB    #使物体活动

create_curve('ha',file_to_pointlist('/home/bacon/a'))

6 Blender绑定

绑定是动画的基础，角色绑定使用骨骼系统，还有物理绑定等

6.1 父子关系

创建绑定的关系：每个选中的物体都是当前物体的子绑定 父物体的移动会导致子物体移动，但子物体不能移动父物体 在属性中可以打开或关闭显示关系线 创建空物体，只是一个参考点，显示形状可以设置,利用空物体将各种东西分组控制，空物体可以设置显示名称 快捷键：

快捷键 功能
Alt+G 回到原点（创建父子关系时的位置）
alt+P 清除父子关系
Ctrl+P 创建绑定

6.2 骨骼系统

Shift+A 创建一个骨骼，骨骼分头和尾，原点就是头，用e自动延伸骨骼并创建父子关系 通过使物体成为骨骼的子物体来进行控制，在Ctrl+P中选择骨骼绑定 骨骼选项中可以设置是否继承旋转、比例等属性 在姿势模式下创作动画，而不是编辑模式或物体模式

6.3 约束

在右侧里创建各种约束，对骨骼要在骨骼页里创建约束 约束的顺序从上到下，最上面的先起作用，可以覆盖下面的 追踪约束，可以使一个物体指向另一个物体 曲线追踪约束，让物体只沿着贝塞尔曲线运动 变形约束，映射变形，可以将一个物体的平移运动与另一个物体的旋转进行绑定转换

6.4 角色绑定，让角色动起来

在设置中选中Rigify，这样可以进行元绑定，创建一个人体骨骼，缩放和旋转骨骼使骨骼和模型相重合（打开X轴镜像再编辑能节省工作量） 不需要的骨骼就删除掉，关节骨骼要在皮肤表面，这样动作更准确，线框模式并打开X光选项，这样就能在物体模式下看见里面的骨骼了 创建父子关系，在Ctrl+P中选择自动权重，Ctrl+TAB进入权重图模式，在T菜单里选上权重刷，打开镜像，权重越大变形越严重，用权重刷刷上0可以消除形变 骨骼形变修改器和其他的修改器，镜像等，顺序很重要，会影响行为 反动力与正动力，IK与FK，让角色可以自然的接触地面 FK就是一般的层级结构，摆姿势时用自动IK功能可以方便些 IK约束，复制一个不可见的子骨骼，删除父子关系，再用新建的这个创建IK约束，避免父子关系死循环，设置正确的极点角度来得到合适的控制效果 姿势模式下也有层级和组功能，能够隐藏不想控制的骨骼，降低复杂度
