Tkinter笔记

基本概念

安装与使用

  1. 不用安装，标准自带
  2. 使用 
import tkinter as tk

运行逻辑

  1. 定义tk.Tk()的实例ROOT ROOT=tk.Tk()
  2. 生成软件 APP=Application(master=ROOT)
  3. 运行主循环APP.mainloop()

布局

注意在一个软件内只能使用同一种布局，且每个继承都要写明布局

pack

功能 属性 说明
默认状态   从上到下顺次排放, 并水平居中

|和其父窗口一样宽 | w.pack(fill=X) || |设置水平方向的外边距 | padx=10 || |设置竖直方向的外边距 | pady=10 || |设置水平方向的内边距 | ipadx=10 || |设置竖直方向的内边距 | ipady=10 ||

顺次放置控件 side=LEFT LEFT RIGHT

grid

功能 属性 说明

|行号 | row=0 || |列号 | column=2 || |跨列 | columnspan=2 || |跨行 | rowspan=2 || | | sticky=W+E+N+S || |外边框x | padx=5 || |外边框y | pady=5 ||

分割线Separator

  1. vertical 垂直线
  2. horizontal 水平线

s = ttk.Separator(parent, orient=HORIZONTAL)

带标签的框Label Frames

lf = ttk.Labelframe(parent, text='Label')

可拖动改变的标签窗口Paned Windows

p = ttk.Panedwindow(parent, orient=VERTICAL)
= first pane, which would get widgets gridded into it:=
f1 = ttk.Labelframe(p, text='Pane1', width=100, height=100)
f2 = ttk.Labelframe(p, text='Pane2', width=100, height=100)   = second pane=
p.add(f1)
p.add(f2)

选择显示的标签页Notebook

n = ttk.Notebook(parent)
f1 = ttk.Frame(n)   = first page, which would get widgets gridded into it=
f2 = ttk.Frame(n)   = second page=
n.add(f1, text='One')
n.add(f2, text='Two')
