ES6基础

基本语法

let x=1 声明变量,变量在作用域内    
const a=1 声明常量,不可变    
+,,-,*,/,% 基本数字运算    

|&&,||,!| 基本逻辑运算|

"a"+"b" 字符串连接    
===,!== 比较运算    
Object.is(a,b) 值相同就相等    
typeof x 返回数据类型    
`x${varname}b` 字符串格式化,使用反引号    
"s".length 字符串长度    
"s".indexOf("s") 子字符串搜索    
"s".slide(0,1) 字符串分割,提取子字符串    
if (){}else if(){}else{} if要加括号    
()?{}:{} 三元运算    
function fun(x="a", ...res){} 定义函数,可加默认值和剩余参数    
(x,y)=>{} 箭头函数,匿名函数    
foo::bar===bar.bind(foo) 函数环境绑定,提供了指定的上下文this    
for (let x=1;x>1;x++){} 普通for循环    
for (let i of b) 迭代器for遍历    
break,continue,do-while,while 和c一样    
let a={a:"a"} 对象    
a.a===a['a'] 对象属性    
class O extents P{constructor(){this.a};set a(){};get a(){};static _a(){}} 新语法定义对象,继承,构造,设定,取值,静态方法    
Object.assign({},{}) 复制对象    
try {throw new Error('a')}catch(e){} 错误处理    
fetch('url',{method:'GET'}).then(()=>{}).catch(()=>{}) 替代AJAX的fetch    
import {a,b} from 'a.js' 引入模块    
export function a(){} 公开函数,可被引入    
function a(v,callback){callback(v)} 回调函数定义    
Promise((resolve,reject)=>{}).then((val)=>{}).catch((err)=>{}) Promise定义,包括成功返回resolve,失败返回reject,成功回调处理then,失败回调处理catch
