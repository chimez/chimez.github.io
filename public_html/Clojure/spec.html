spec

clojure.spec为clojure提供了一个可选的类型系统，用来定义数据结构和类型，并进行校验，可以补足Clojure作为动态类型语言的一些缺陷，特别是在传递参数时。

安装与应用

clojure.spec是标准库的一部分，在1.9版本正式引入，所以不需要安装依赖 引用格式为： 

(require [clojure.spec.alpha :as s])

基本使用

定义规范

使用s/def定义一个规范，规范函数接受参数作为输入值并返回一个布尔值, 被定义的符号用::表示当前命名空间，为了避免冲突可以用自定义的不真实存在的命名空间:a/b 

(s/def ::even-val even?)

规范函数可以使用组合条件s/and和s/or 

(s/def ::or-test
 (s/or :int-number int?
  :string-number string?))

在spec中有一些内置的规范函数可以使用 s/valid?判断给定数据的是否满足当前的规范 

    (s/valid? ::even-val 10)
    ;;=> true
    (s/valid? ::even-val 11)
    ;;=> false

s/conform根据给定的规范和数据进行解析，如果不符合就返回s/invalid s/explain用来解释匹配失败的原因 对于数组可用的函数有：s/coll-of, s/every, s/tuple 对于map可用的函数有：s/map-of, s/every-of, s/keys 对于nil的处理：s/nilable表示可以传入nil
