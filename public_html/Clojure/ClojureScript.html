ClojureScript 笔记

ClojureScript基本流程

安装

  1. 不需要安装什么东西，只需要一个cljs.jar文件即可
  2. java版本>8

项目目录结构

	源文件：/src/core.cljs

		(ns hello-world.core) ;先声明名字空间
		(enable-console-print!) ;启用控制台输出
		(println "Hello world!")  ;在控制台里输出文字

编译方法

基本编译

在项目目录里创建文件build.clj 

	(require 'cljs.build.api)
		(cljs.build.api/build "src" {:output-to "out/main.js"})')

执行编译命令: 

Linux:
	java -cp cljs.jar:src clojure.main build.clj 

		Windows:
		java -cp "cljs.jar;src" clojure.main build.clj

注意linux里是冒号无引号，windows里是分号有引号 

	<html>
		<body>
		<script type="text/javascript" src="out/goog/base.js"></script>
		<script type="text/javascript" src="out/main.js"></script>
		<script type="text/javascript">
		goog.require("hello_world.core");
	// Note the underscore "_"!
	</script>
		</body>
		</html>

html模板里要引用Google closure library， 编译的文件 并且还要执行js代码

另一种编译方法：减少模板

在build.clj文件中指定主函数 

	(require 'cljs.build.api)

		(cljs.build.api/build "src"
		 {:main 'hello-world.core
		 :output-to "out/main.js"})')

现在html模板可以写成 

	<html>
		<body>
		<script type="text/javascript" src="out/main.js"></script>
		</body>
		</html>

会自动引用包并执行入口函数

自动编译

可以实现监测源文件修改并自动编译,创建watch.clj文件 

	(require 'cljs.build.api)

		(cljs.build.api/watch "src"
		 {:main 'hello-world.core
		 :output-to "out/main.js"})')

执行自动编译 

	java -cp cljs.jar:src clojure.main watch.clj

REPL

在控制台启动REPL的方法，创建文件repl.clj 

	(require 'cljs.repl)
		(require 'cljs.build.api)
		(require 'cljs.repl.browser)

		(cljs.build.api/build "src"
		 {:main 'hello-world.core
		 :output-to "out/main.js"
		 :browser-repl true
		 :verbose true})

		(cljs.repl/repl (cljs.repl.browser/repl-env)
		 :watch "src"
		 :output-dir "out")')

运行命令 

	java -cp cljs.jar:src clojure.main repl.clj

为了增强repl的功能，可以使用rlwrap 

	rlwrap java -cp cljs.jar:src clojure.main repl.clj

运行命令之后需要用浏览器连接后端，打开 http://localhost:9000

自动编译过程的记录：tail -f out/watch.log

发行编译：压缩到一个文件里

创建 release.clj 

	(require 'cljs.build.api)

		(cljs.build.api/build "src"
		 {:output-to "out/main.js"
		 :optimizations :advanced})

		(System/exit 0)')

运行编译：java -cp cljs.jar:src clojure.main release.clj
