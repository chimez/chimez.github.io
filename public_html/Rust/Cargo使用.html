Cargo

基本命令

命令 作用
cargo new hello --bin 创建新项目同时创建了cargo.toml和git仓库
cargo build 编译项目
cargo run 编译并运行项目
cargo build --release 优化编译
cargo test 运行测试
cargo doc --open 自动生成文档网页
cargo install 安装包或者扩展

配置文件

name 项目名
version 版本号
authors 作者信息
dependencies 依赖库,版本号表示>=指定小版本
opt-level 优化级别,不优化为0,最大优化为3
workspace 工作空间
member 子项目

配置文件Cargo.toml写法: 

[package]
name = "gb"
version = "0.1.0"
authors = ["chimez <chimez@163.com>"]

[dependencies]
rand = "0.3.14"
adder = { path = "../add-one" }	//子项目依赖

[profile.dev]
opt-level = 0

[profile.release]
opt-level = 3

[workspace]
members = [
    "adder",
]

目录结构 

├── Cargo.lock
├── Cargo.toml
├── adder
│   ├── Cargo.toml
│   └── src
│       └── main.rs
└── target
