pyo3

pyo3是rust和python之间的绑定库,既可以从rust使用python函数也可以从python使用rust

基本使用

python调用rust函数

cargo配置

编译后的文件名为libname.so,需要手动改成name.so,这样才能从python中直接引用 

[lib]
crate-type = ["cdylib"]
[dependencies.pyo3]
version = "0.2"
features = ["extension-module"]

编写rust





#![feature(proc_macro, specialization)]


使用预编译宏的前提






#[py::modinit(hello)]


导出mod






#[pyfn(m, "run_rust_func")]


导出函数





#![feature(proc_macro, specialization)]
extern crate pyo3;
use pyo3::{py, Python, PyResult, PyModule, PyString};
#[py::modinit(hello)]
fn init_module(py: Python, m: &PyModule) -> PyResult<()> {
    #[pyfn(m, "run_rust_func")]
    fn run(name: &PyString) -> PyResult<()> {
        println!("Rust says: Hello {} of Python!", name);
        Ok(())
    }
    Ok(())
}

编写python

使用函数:只要把so放进路径里就能直接用了 

import hello
hello.run_rust_func()

rust调用python函数

cargo配置

cargo init --bin中配置上依赖 

[dependencies]
pyo3 = "0.2"

编写rust

无需特殊语法,只用引入python函数 

extern crate pyo3;

use pyo3::{Python, PyDict, PyResult, ObjectProtocol};

fn main() {
    let gil = Python::acquire_gil();
    hello(gil.python()).unwrap();
}

fn hello(py: Python) -> PyResult<()> {
    let sys = py.import("sys")?;
    let version: String = sys.get("version")?.extract()?;

    let locals = PyDict::new(py);
    locals.set_item("os", py.import("os")?)?;
    let user: String = py.eval("os.getenv('USER') or os.getenv('USERNAME')", None, Some(&locals))?.extract()?;

    println!("Hello {}, I'm Python {}", user, version);
    Ok(())
}
