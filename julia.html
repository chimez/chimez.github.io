1 Julia 学习笔记

1.1 基本功能

1.1.1 变量与赋值

  1. 变量名用 δ + tab 可以生成希腊字母
  2. 赋值就是等号 a=1
  3. 语句不用分号,但同一行内可以用来进行分隔
  4. 查看类型用 typeof(a)
  5. 用 Inf 表示无穷
  6. 作用域可以用 global a local a outer a
  7. 声明常量 const a =1

1.1.2 运算符

  1. 乘方用 1^2
  2. 乘法可以略去 1(2) == 1*2
  3. 取反用 !a
  4. 除法的 1/2 == 2\1
  5. 不等于号 1!=2
  6. 按位取反是波浪线
  7. 和 matlab 类似有点操作，即每个元素操作 [1 2] .+ 2

1.1.3 数据类型

  1. 基本数据类型
    1. 复数用 1+2im
    2. 两个除号可以表示一个分数，这个分数不会被除成浮点数 1//2
    3. 字符用单引号，字符串双引号，多行字符串用三双引号
    4. 字符串内可执行语句 "1+1 = $(1+1)"
    5. 矩阵的行内用空格，换行用逗号或者分号 [1 2;3 4]
    6. 下标从1开始
    7. 元组用括号 (1,2,3,)
  2. 类型系统
    1. 类型声明用双冒号 (1+2)::AbstractFloat
    2. 类型判断用 isa isa(("1",), mytupletype)
    3. 定义新的类型, 不指定父类型的话是 Any 类型 abstract type Number end
    4. 指定父类型用 <: abstract type Real <: Number end
    5. 定义原始类型, 由上级类型加上字节数来定义 primitive type Float16 <: AbstractFloat 16 end
    6. 类型别名直接赋值就行 UInt = UInt64
    1. 复合类型

      1. 定义复合类型 struct 

         struct Foo
            bar
            baz::Int
            qux::Float64
        end

      2. 定义可变复合类型 mutable struct 

         mutable struct Bar
           baz
           qux::Float64
       end
      3. 联合类型 Union IntOrString = Union{Int,AbstractString}

      4. 复合类型可以泛型 

        struct Point{T}
           x::T
           y::T
       end

      5. 函数参数中指定复合类型的类型 

        function norm(p::Point{<:Real})
    sqrt(p.x^2 + p.y^2)
end

      6. 类型用大括号,参数用小括号 

        Point{Float64}(1.0, 2.0)
      7. 可以定义基本抽象类型 abstract type Pointy{T} end

      8. 或者好几个类型的泛型 Tuple 类型 

        struct Tuple2{A,B}
    a::A
    b::B
end

      9. 也能指定类型的名字 

        NamedTuple{(:a, :b),Tuple{Float32, String}}((1,""))
      10. 基本类型的泛型 primitive type Ptr{T} 32 end
      11. 类型的声明可以放在后面 Tuple{T,Array{S}} where S<:AbstractArray{T} where T<:Real
    2. 值类型

      类似关键字 

       struct Val{x}
       end
firstlast(::Val{true}) = "First"
firstlast(Val(true))
"First"

1.1.4 正则表达式

  1. 查找 match(r"^(.+)", "## a comment")
  2. 替换 replace("a", r"." => s"\g<0>1")

1.1.5 函数

  1. 定义函数
    function f(x,y)
           x + y
end

    或者 

    f(x,y) = x + y
  2. 返回值与类型注解

    可以有多重返回值 

    function g(x, y)::Int8
           return x * y, x-y
       end;

    可选参数用等号 

    function Date(y::Int64, m::Int64=1, d::Int64=1)
    err = validargs(Date, y, m, d)
    err === nothing || throw(err)
    return Date(UTD(totaldays(y, m, d)))
end
  3. lisp 风格
    +(1,2,3)
1 + 2 + 3
  4. 匿名函数
    x -> x^2 + 2x - 1
  5. 关键字参数

    用分号分开 

    function plot(x, y; style="solid", width=1, color="black")
    ###
end

    调用时 

    plot(x, y; :width => 2)

    或者 

    plot(x, y, width=2)
  6. 对矩阵参数的每个元素应用函数

    在函数后面加上点 

    Y = [1.0, 2.0, 3.0, 4.0];
X .= sin.(cos.(Y))

    或者在前面 

    @. X = sin(cos(Y))
  7. 连续应用函数 |>
    [1:5;] .|> [x->x^2, inv, x->2*x, -, isodd]

1.1.6 方法

  1. 定义方法

    1. 像函数那样定义 

      f(x::Number, y::Number) = 2x - y

    2. 查看函数实现在了哪些上 

      methods(f)

    3. 参数的方法 

      julia> same_type(x::T, y::T) where {T} = true
same_type (generic function with 1 method)

julia> same_type(x,y) = false
same_type (generic function with 2 methods)

1.1.7 逻辑结构

  1. 语句块

    就是大括号,最后一个执行的就是返回值 

        begin 
         1+1
    end

    也可以用小括号 

    (x = 1;
        y = 2;
        x + y)

    定义一个变量的语句块 

    let i = i
               Fs[i] = ()->i
           end

    又如 

    let
           local x = 1
           let
               local x = 2
           end
           x
       end
  2. do 结构
    1. 用来把 map 里面的逻辑放到外面去
       map(x->begin
            if x < 0 && iseven(x)
                return 0
            elseif x == 0
                return 1
            else
                return x
            end
        end,
     [A, B, C])

       map([A, B, C]) do x
     if x < 0 && iseven(x)
         return 0
     elseif x == 0
         return 1
     else
         return x
     end
 end
    2. 用来读写文件
      open("outfile", "w") do io
    write(io, data)
end
  3. 条件判断
    if x < y
    println("x is less than y")
elseif x > y
    println("x is greater than y")
else
    println("x is equal to y")
end
  4. 三目运算符和短路运算
    a ? b : c
a && b
a || b
  5. 循环
    1. while
       i=1
   while i <= 5
             println(i)
             global i += 1
         end
    2. for
      1. 数字区间循环
          for i = 1:5
            println(i)
        end
      2. 遍历列表
         for i in [1,4,0]
            println(i)
        end

          for s ∈ ["foo","bar","baz"]
            println(s)
        end
      3. 二重循环
        for i = 1:2, j = 3:4
           println((i, j))
           i = 0
       end
(1, 3)
(1, 4)
(2, 3)
(2, 4)
  6. 异常
    1. 自定义异常
      struct MyCustomException <: Exception end
    2. 抛出异常
      throw(DomainError(x, "argument must be nonnegative"))
    3. 异常处理
      sqrt_second(x) = try
           sqrt(x[2])
       catch y
           if isa(y, DomainError)
               sqrt(complex(x[2], 0))
           elseif isa(y, BoundsError)
               sqrt(x)
           end
       end
    4. 终结操作
      f = open("file")
try
    # operate on file f
finally
    close(f)
end
