R语言画图

R是统计分析和绘图的专用语言,matplotlib画2D图像确实很好,也足够了,但是3D就不行了,尤其是想像matlab那样画完3D还能转,mayavi和pyqt复杂度都异常的高,不如换R来得方便.

数据交互

从Python将数据导出,再从R中引入,方法有很多,还有各种专用库,其中成本最小的是导出成csv: 写入csv: 

import csv
data = [[1,2],
				[2,3]]
with open('gk.csv','w') as fi:
    writer = csv.writer(fi)
    for row in data:
        writer.writerow(row)

读取csv: 

with open('gk.csv') as f:
    reader = csv.reader(f)
    print(list(reader))

R中读取CSV: 

data1<-read.table('filename',sep=',')#默认header=FALSE

R中3D画图--rgl

  • 首先安装rglinstall.apckage('rgl')
  • 有可能出现问题,自行apt-file search搜索安装
  • 会提示缺少各种包,自行安装

画三维散点图并旋转加保存到文件

data1<-read.table('/home/bacon/Documents/Spin Squeezing/gk.csv',sep=',')#默认header=FALSE
library(rgl)
open3d()
x<-data1$V1
y<-data1$V2
z<-data1$V3
plot3d(x,y,z,col=rgb(0.3,0.3,1),
       xlab = "g1",ylab = "kappa",zlab = "co"#,
       #xlim = c(0,1), ylim = c(0,1), zlim = c(0,1)
       )
movie3d(spin3d(axis=c(0,0,1),rpm=7),duration=30,fps=25,movie = "~/Documents/gk")
