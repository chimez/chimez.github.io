函数

创建函数

创建普通函数
;;    name   params         body
;;    -----  ------  -------------------
(defn greet  [name]  (str "Hello, " name) )
创建多元函数（不同输入对应不同输出）

Each arity is a list ([args*] body*) 

(defn messenger
  ([]     (messenger "Hello world!"))
  ([msg]  (println msg)))
创建不定元函数（参数个数不定）
(defn hello [greeting & who]
  (println greeting who))
创建匿名函数
;;    params         body
;;   ---------  -----------------
(fn  [message]  (println message) )
defn、def、fn之间的关系

defn是(def (fn))的语法糖

语法匿名函数，匿名函数的语法糖
  1. % 表示单个参数
  2. %1, %2, %3 表示第几个参数
  3. %& 表示剩余参数
  4. =() 表示匿名函数= 
=(+ 6 %)=
(fn [x] (+ 6 x))
=(println %1 %2 %&)=
(fn [x y & z] (println x y z))

应用函数 Apply

  1. Apply函数将第一个参数作为函数，将其余全部参数作为函数的参数，执行这个函数
  2. 同时，那些其余参数如果是列表的话，将会解开，将每个元素都作为参数，依次排列下去 
(apply f 1 '(2 3 4))
(f 1 2 3 4)

局部和闭包

let

let会创造一个词汇语义范围，在范围内将值与符号绑定，类似局部变量 

;;      bindings     name is defined here
;;    ------------  ----------------------
(let  [name value]  (code that uses name))
(let [x 1
      y 2]
  (+ x y))

闭包

  1. fn函数所形成的叫做闭包
  2. 闭包包括了周围的语义范围，参数和函数等
  3. 所以在这个闭包内并不能修改全局变量

流程控制

if

(if (true? false) "impossible!")
(if (even? 2) "even" "odd")

只有 false nil是假的

do

 (do (println "even")
      true)

when

(when (neg? x)
  (throw (RuntimeException. (str "x must be positive: " x))))

when 没有else路径

cond

对每个路径都进行判断 

(let [x 11]
  (cond
    (< x 2)  "x is less than 2"
    (< x 10) "x is less than 10"
    :else  "x is greater than or equal to 10"))

case

(defn foo [x]
         (case x
           5 "x is 5"
           10 "x is 10"
           "x isn't 5 or 10"))

有副作用的迭代

dotimes
(dotimes [i 3]

(doseq [n (range 3)]

for

for并不是循环而是几个列表遍历组装新列表 

(for [letter [:a :b]
             number (range 3)] ; list of 0, 1, 2
         [letter number])
([:a 0] [:a 1] [:a 2] [:b 0] [:b 1] [:b 2])

递归

必须使用尾递归而且显式声明 

(loop [i 0]
  (if (< i 10)
    (recur (inc i))
    i))
(defn increase [i]
  (if (< i 10)
    (recur (inc i))
    i))

异常

try-catch捕获异常

(try
  (/ 2 1)
  (catch ArithmeticException e
    "divide by zero")
  (finally
    (println "cleanup")))

try-throw抛出异常

(try
  (throw (Exception. "something went wrong"))
  (catch Exception e (.getMessage e)))

ex-info 将异常信息作为map

  1. ex-info takes a message and a map
  2. ex-data gets the map back out
  3. Or nil if not created with ex-info 
(try
  (throw (ex-info "There was a problem" {:detail 42}))
  (catch Exception e
    (prn (:detail (ex-data e)))))

with-open 写法

(let [f (clojure.java.io/writer "/tmp/new")]
  (try
    (.write f "some text")
    (finally
      (.close f))))

;; Can be written:
(with-open [f (clojure.java.io/writer "/tmp/new")]
  (.write f "some text"))
