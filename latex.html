1 如何修改 REVTeX 的字体与行号

REVTex 是APS论文的指定样式，命令总结可以在这里找到： http://hep.ucsb.edu/courses/ph128_18s/summary4-1.pdf

默认可选 10pt, 11pt, 12pt 三种，但是改字号后行距也会跟着变，有时非常丑，所以，需要同时修改行距。 修改行距使用命令 \linespread{0.956} 。参考 Explicit line spacing in revtex 4

默认行间距大小对照表，参考 Why is the linespread factor as it is?

字号 行间距
10pt 12pt
11pt 13.6pt
12pt 14.5pt

示例： 

\documentclass[11pt,tightenlines]{revtex4}

% Alternative A
% \usepackage{etoolbox}
% \patchcmd{\normalsize}{13.6}{13}{}{}

% Alternative B
\linespread{0.956}

\usepackage{lipsum}

\begin{document}

\lipsum[1]

\end{document}

2 如何配置 CJK

作为中文支持的CJK宏包早就淘汰了，但是国产的垃圾杂志却都还在用，所以还是要折腾。

参考王垠当年的说明。

2.1 配置步骤

2.1.1 1. 下载 gbkfonts 程序

首先去这里下载 gbkfonts 程序。 这个东西都是王垠2002年写的啊，幸亏Windows向后兼容性好，不然的话还真没人能解决得了这个问题。

2.1.2 2. 创建字体文件

命令大概就是下面这几个，程序运行结束后会在同一个目录里创建大量文件夹。 

./gbkfonts.exe -c -gbk ./simsun.ttf  song
./gbkfonts.exe -c -gbk ./simhei.ttf  hei
./gbkfonts.exe -c -gbk ./simkai.ttf  kai
./gbkfonts.exe -c -gbk ./simfang.ttf fs

2.1.3 3. 拷贝文件并刷新数据库

就是把上一步生成的所有文件都复制到 /usr/local/texlive/texmf-local/ 目录，或者 $$TEXMFLOCAL$ 目录。 此外还要移动一些文件, 参考paper 47 ：Latex中文显示(转) 

mv cid-x.map dvipdfm/config/
mv cjk.map dvips/config/
cp /usr/share/texmf/dvips/config/config.ps dvips/config/
echo 'p +cjk.map' >> dvips/config/config.ps
mv pdftex.cfg pdftex/config/
mktexlsr

之后在这个目录里运行 mktexlsr 刷新文件名数据库。

2.1.4 4. 测试

复制完成后应该就能用了，示例文件如下： 

% !TeX encoding = GBK

\documentclass{article}
\usepackage{CJK}
\begin{document}
    \begin{CJK}{GBK}{song}
    \LaTeX 测试
    \end{CJK}
\end{document}

编译必须用 latex 然后 dvipdfm 得到 pdf 文件。不知道为什么不能用 pdflatex 

latex document.tex
dvipdfm document.dvi

2.2 其他说明

2.2.1 ttc与ttf格式

ttf格式是TrueType类型字体，是通用的标准格式。 而新版的windows系统中的字体是ttc格式。 ttc格式是微软开发的新一代格式，基本上相当于几个ttf格式的组合体。

2.2.2 文件

所需要的文件都存在 Dropbox/Programs/latex/gbkfonts 里

5 配置与安装

5.1 使用 latexmk 自动执行编译

5.1.1 解释

$pdf_mode = 1;

默认使用 pdflatex 命令，下面重载命令的内容。 

$pdflatex = "xelatex -file-line-error -halt-on-error -interaction=nonstopmode --shell-escape -src-specials -synctex=1 %O %S;cp %D %R.pdf";

实际使用 xelatex 命令，其中

  1. -file-line-error 使报错输出文件和行号；
  2. -halt-on-error 和 -interaction=nonstopmode 使编译遇到错误时立即停止；
  3. -synctex=1 则开启 synctex 的功能;
  4. --shell-escape 关闭 \write18;
  5. -src-specials 向 xdv 中插入代码;
  6. %O 选项
  7. %S tex 源文件
  8. %D 目标文件
  9. %R 根文件名，即源文件去掉后缀
$recorder = 1;

传递 -recoder 选项，生成 fls 文件，追踪生成文件的过程。 

$pdf_previewer = "xdg-open %S";

$pdf_update_method = 0;

pdf 文件改变时让阅读器改变 

$clean_ext = "synctex.gz acn acr alg aux bbl bcf blg brf fdb_latexmk glg glo gls idx ilg ind ist lof log lot out run.xml toc dvi";

需要清理的后缀文件名 

$bibtex_use = 1.5;

自动决定运行一遍还是两遍 bibtex 

$out_dir = "temp";

把所有中间文件都放到 temp 文件夹里

5.1.2 完整脚本

$pdf_mode = 1;
$pdflatex = "xelatex -file-line-error -halt-on-error --shell-escape -src-specials -synctex=1 -interaction=nonstopmode %O %S;cp %D %R.pdf";
$recorder = 1;
$pdf_previewer = "xdg-open %S";
$pdf_update_method = 0;
$clean_ext = "synctex.gz acn acr alg aux bbl bcf blg brf fdb_latexmk glg glo gls idx ilg ind ist lof log lot out run.xml toc dvi";
$bibtex_use = 1.5;
$out_dir = "temp";

6.1.4 语句选项

语法： 

\draw[dashed, line width=2] (1,2)--(3,4);

参数

line width=1pt 线宽
thin, very thin, ultra thin, thick, very thick, ... 预定义线宽
dashed pattern=on 2pt off 3pt on 4pt 手动设置线条形状，on、off表示画线不画线的长度，重复出现
dashed phase=1pt 设置第一个虚线缩短的长度
color=颜色 设置线色
draw=颜色 设置线色
=fill=green!20` 填充颜色，这里表示20%的绿色，后面接着写`red,blue,white= 可以混合颜色
opacity=0.5 设置透明度
rounded corners=10pt 设置拐点圆角半径，默认为0
scale=0.3 放大缩小倍数
double 画双线
double distance=1pt 双线间距离
left, right, top, bottom, inner, outer, ball 阴影的渐变方向
