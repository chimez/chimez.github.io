4.1 sed

4.1.1 sed 正则表达式

字符 功能
- -
^ 行首定位
$ 行尾定位
\< 词首定位
\> 词尾定位
. 匹配一个字符
* 匹配多个字符
[] 匹配一组字符
[x-y] 匹配范围
[^] 匹配范围外
x\{m,n\} 字符x的m到n次
\ 转义
\(...\) 标签

4.3 fd 代替 find

4.3.1 安装

      cargo install fd-find --force # 替代 find

4.3.2 使用

      # 显示全部文件

      # 查找文件

5 anbox: 在 Linux 上运行 Android 应用

5.1 安装 anbox

  1. 在源里安装 apt install anbox
  2. 阅读说明： /usr/share/doc/anbox/README.Debian
  3. 下载 android 镜像 https://build.anbox.io/android-images
  4. 放在 /var/lib/anbox/android.img
  5. 启动系统服务（root 权限） service anbox-container-manager start
  6. 启动服务，普通用户： anbox session-manager

  7. 完成，可以在桌面上点开应用了

    只能安装 x86 的应用，实在是太少了
